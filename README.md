# My Community Forum – Social Development Events Platform

**Author:** Al-Roman Molla

**My Community Forum** is a community-driven event management platform that enables users to **create, join, and track social service events** in their local area. The platform encourages community engagement in meaningful activities such as road cleaning, tree plantation, and donation drives, helping users make a tangible impact in their neighborhoods.

**Live Site:** [https://mycommunityforum.web.app/](https://mycommunityforum.web.app/)  
**GitHub Repository (Client):** [Frontend Repository](https://github.com/Al-Roman23/My_Community_Forum_Frontend)  
**GitHub Repository (Server):** [Backend Repository](https://github.com/Al-Roman23/My_Community_Forum_Backend)

## Key Features

- **User Authentication:** Secure email/password login and registration with Google login support. Password validation and real-time feedback ensure a reliable authentication experience.
- **Event Creation:** Authenticated users can create events with essential details, including title, description, event type, location, thumbnail, and event date. Only future dates can be selected to maintain event relevance.
- **Upcoming Events:** Publicly accessible page showcasing all upcoming events in an organized grid layout, complete with event thumbnails, titles, locations, types, and dates.
- **Event Participation:** Users can join events once logged in, with participant data stored in the database for proper tracking.
- **Event Management:** Creators can view, update, and optionally delete their events through a dedicated management dashboard.
- **Private Routes:** Pages like `Create Event`, `Manage Events`, and `Joined Events` are accessible only to authenticated users, ensuring secure access.
- **Home Page Sections:** Fully designed Banner, Features, Gallery, and Newsletter sections tailored to the platform’s theme.
- **Theme Toggle:** Seamless light and dark mode switching across the platform for a personalized user experience.
- **Search and Filter:** Users can filter events by type and search by name, allowing quick access to relevant events.
- **Responsive Design:** Optimized for all devices, including desktops, tablets, and mobile phones.
- **Animations:** Smooth and engaging UI animations implemented using Framer Motion for better interactivity.
