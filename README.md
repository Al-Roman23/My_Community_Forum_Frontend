## Project Overview
**My Community Forum** is a **community-driven event management platform** that enables users to create, join, and track social service events in their local area. The platform promotes **community engagement** through activities like road cleaning, tree plantation, and donation drives, allowing users to make a tangible impact in their neighborhoods.

Built with **React.js, Next.js, Firebase, and MongoDB**, this project demonstrates **frontend-focused web development** principles, secure authentication, responsive design, and production-ready practices.

---

## Key Features

- **User Authentication:** Email/password login, Google login, and password validation.  
- **Event Creation:** Authenticated users can create events with title, description, type, location, thumbnail, and future date selection.  
- **Upcoming Events:** Browse all upcoming events in a clean, organized grid layout.  
- **Event Participation:** Join events and track participants.  
- **Event Management Dashboard:** Update, manage, or delete events securely.  
- **Private Routes:** Pages like `Create Event`, `Manage Events`, and `Joined Events` require authentication.  
- **Home Page Sections:** Banner, Features, Gallery, and Newsletter.  
- **Theme Toggle:** Light and Dark mode support.  
- **Search & Filter:** Filter events by type and search by name.  
- **Responsive Design:** Optimized for desktop, tablet, and mobile.  
- **UI Animations:** Smooth interactions powered by Framer Motion.
